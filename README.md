# Python Quiz Game
A terminal-based quiz game built with Python featuring multiple categories, randomised questions, score tracking, replay functionality, and performance-based feedback.

## Features
- Multiple Quiz categories:
  - Python
  - Maths
  - Harry Potter
- Randomised question order 
- Score calculation
- Performance-based feedback
- Replay option

## Project Structure
```text
quizgame.py
ques_python.py
ques_maths.py
ques_harry.py
```

## About
This project was created while learning Python and practicing:
- Dictionaries
- Loops
- Conditional statements
- Modules and imports
- Randomization
